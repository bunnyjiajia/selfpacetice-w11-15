1 โค้ดเริ่มต้นด้วยการรอให้หน้าเว็บโหลดเสร็จก่อน โดยใช้ DOMContentLoaded เพื่อให้มั่นใจว่าองค์ประกอบทั้งหมดในหน้า HTML ถูกสร้างเรียบร้อยแล้วก่อนเริ่มทำงานของ JavaScript
2 เมื่อหน้าเว็บโหลดเสร็จ โค้ดจะเลือกปุ่ม ช่องค้นหา พื้นที่ข้อความ และรายการผู้ใช้จาก HTML เพื่อนำไปใช้งาน
3 มีตัวแปรชื่อ users สำหรับเก็บข้อมูลผู้ใช้ทั้งหมดที่ได้รับจาก API เพื่อใช้แสดงผลและค้นหาได้ภายหลัง
4 เมื่อผู้ใช้กดปุ่มโหลดข้อมูล ฟังก์ชัน loadUsers() จะทำงาน โดยจะเปลี่ยนข้อความสถานะเป็น “กำลังโหลดข้อมูล” และล้างรายการเก่าออกก่อน
5 ฟังก์ชันดังกล่าวเรียก API จริงด้วย fetch() และใช้ await เพื่อรอให้ข้อมูลตอบกลับ ก่อนอ่านผลลัพธ์ด้วย response.json()
6 ถ้า API ตอบกลับด้วยสถานะที่ไม่ปกติ (ไม่ใช่รหัส 200) โค้ดจะเข้าสู่ส่วนจับข้อผิดพลาด (catch) และแสดงข้อความแจ้งเตือน
7 เมื่อโหลดข้อมูลสำเร็จ รายชื่อผู้ใช้จะถูกแสดงใน <ul> โดยสร้าง <li> ใหม่ตามจำนวนข้อมูลที่ได้รับ
8 ช่องค้นหาจะถูกแสดงหลังจากโหลดข้อมูลสำเร็จ เพื่อให้ผู้ใช้สามารถพิมพ์คำค้นหาได้
9 ฟังก์ชัน filterUsers() จะทำงานทุกครั้งที่มีการพิมพ์ในช่องค้นหา โดยจะกรองรายชื่อที่ตรงกับข้อความค้นหาและแสดงผลที่กรองแล้วทันที
10 ฟังก์ชัน renderList() ถูกใช้เพื่ออัปเดตรายการในหน้าเว็บทุกครั้ง ทั้งตอนแสดงข้อมูลครั้งแรกและตอนกรองข้อมูล
11 ปุ่มโหลดข้อมูลและช่องค้นหาถูกผูกเหตุการณ์ (addEventListener) ไว้ให้เรียกใช้ฟังก์ชันที่เกี่ยวข้อง
